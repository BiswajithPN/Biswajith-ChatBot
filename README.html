Quick Setup

1
Clone or Download the Repository
git clone <repository-url>
2
Install Required Dependencies
pip install -r requirements.txt
3
Configure Environment Variables
# Create a .env file in the project root
GROQ_API_KEY=your_groq_api_key_here
4
Run the Application
streamlit run biswajith_chatbot.py

How to Use

  • Start Chatting: Simply open the application in your browser and begin typing your questions or prompts
  • API Key Setup: Ensure your Groq API key is properly configured in the .env file
  • Real-Time Responses: Receive instant, intelligent responses powered by advanced AI
  • Professional Interface: Enjoy a sleek, modern interface optimized for productivity
  • Customizable: Easily modify styling and configurations to match your preferences
